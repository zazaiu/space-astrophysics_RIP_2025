# Разработка интернет-приложений (РИП) 2025

##Расстояние 8 планет солнечной системы от Солнца и его угол между текущим положением и перегелием для указанной даты.

Система обеспечивает создание и модерацию заявок на расчёт астрономических параметров: расстояния от Солнца и угла между текущим положением планеты и её перигелием для указанной пользователем даты.
